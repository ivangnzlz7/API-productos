### Una api de productos de computación

# Instalacion



## 1.En la terminal, clone el repositorio con el comando git clone 'url'
## 2.Crea una variable de entorno dentro de la carpeta que clonamos con python3/python -m venv venv
## 3.Activa el entorno con source venv/bin/activate (Linux)
## 4.Instale las dependencias con pip install -r requirements.txt
## 5.En la terminal use flask --app index run para que corra el servidor
